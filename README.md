# ELP_predction_ML
This study presents a machine learning model using Random Forest and SVM algorithms to predict the winner of the English Premier League (EPL). The model achieved an accuracy score of 0.79 on the test data, indicating its effectiveness in predicting the EPL winner. The dataset used for training and testing the model consisted of 2600 matches from 2014 to 2021. The results demonstrate that the proposed model can accurately predict the winner of the EPL and could be used by football clubs, sports analysts, and enthusiasts to make informed decisions. The findings of this study could have significant implications for the football industry, as well as for sports analytics and betting.
